# CyberBook — распределение задач (хакатон, кейс МТС)

**Продукт:** корпоративный ассистент по кибербезопасности. Ядро — AI-тренер на
Cerebras (gpt-oss-120b), который обучает, генерирует квизы/фишинг, разбирает ошибки.

**Стек:** Python Flask + SQLite (бэкенд), vanilla JS/HTML/CSS (фронт). Один сервер.

**Что УЖЕ готово (ядро подняли):** auth, модели, AI-модуль, фишинг-движок,
скоринг/бейджи, база курсов/квизов/статистики, демо-данные, логин + дашборд сотрудника.
Всё запускается и протестировано. Расширяем и наполняем.

## 🟣 Фронтенд — vanilla JS, страницы в `static/`
Готово: `index.html` (логин/регистрация), `dashboard.html` (обзор + score). Используй
`static/js/api.js` (готовый хелпер) и `static/css/style.css` (брендинг МТС). Осталось создать:

1. [ ] **courses.html** — каталог курсов (`API.courses()`) + страница урока (`API.course(id)`):
       текст (content), встроенное видео (video_url — iframe), кнопка «пройти квиз».
2. [ ] **quiz.html** — квиз (`API.quiz(id)`): вопросы с вариантами → `API.submitQuiz(id, answers)` →
       экран результата с разбором (review) и подсветкой правильных/ошибок.
3. [ ] **inbox.html** — 📨 фишинг-тренажёр (`API.inbox()`): список писем как почта, открытие письма,
       кнопки «Сообщить о фишинге / Доверять» → `API.answerPhishing(id, action)` →
       показать correct + red_flags + AI-разбор. **Killer-фича для демо.**
4. [ ] **assistant.html** — 🤖 чат с AI (`API.chat(message, history)`): пузыри сообщений,
       индикатор набора. **Вторая killer-фича.**
5. [ ] **leaderboard.html** — 🏆 рейтинг (`API.leaderboard()`): таблица с местами, score, бейджами.
6. [ ] **admin.html** — админ-дашборд (`API.overview()`): карточки-метрики, срез по отделам,
       «в зоне риска», кнопка «запустить фишинг-кампанию» (`API.generatePhishing()`).

Навигация уже есть в шапке `dashboard.html` — копируй её на другие страницы.
